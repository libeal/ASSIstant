ast+rules

文件变更重定向、cp/mv/rm/tee/rsync/chmod/chown 等必须走受控 Skill。
隐藏执行命令替换、进程替换、shell -c、xargs/timeout 等转发器需要审查。
数据外传curl/wget 上传本地内容、file:// 读取等进入人工审批。
混淆检测识别 ${IFS} 等分词混淆后再做远程管道和命令头审查。
交互命令watch、less、vim、tail -f、未带批处理参数的 top 不适合 agent 执行。

正则审查表

阻断规则、警告规则和原因一屏可查；解锁后可编辑。

级别正则 / 关键词动作原因
blockrm\s+-rf\s+/禁止执行阻止根目录递归删除。
blockcurl\s+.*\|\s*sh禁止执行禁止流式远程脚本执行。
blockchmod\s+-R\s+777\s+/禁止执行阻止破坏性权限变更。
